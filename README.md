# Registration_Form
The project creates a basic Registration Form through the use of HTML, CSS, and JavaScript. The form collects user details such as Name, Email, and Password and performs client-side validation to ensure correct input before submission.

# Features

Validates that the Name field is not empty

Checks whether the Email is in a valid format

Ensures the Password contains at least 6 characters

Displays error messages for invalid inputs

Submit button remains disabled until all fields are valid

Uses JavaScript DOM manipulation and event listeners for validation

# Technologies Used

HTML

CSS

JavaScript

# Purpose

The main goal of this project is to practice form validation, DOM manipulation, and user input handling in JavaScript.

# Output

When users enter valid information in all fields, the Submit button becomes enabled, and the form can be successfully submitted.
